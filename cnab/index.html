Dicas para colar

  • Não inclua a linha de cabeçalho — apenas os dados.
  • Valores podem ter espaços e vírgula decimal: 25,38, 1.250,00 ou "R$ 61,00".
  • Documentos serão validados como CPF (11 dígitos) ou CNPJ (14 dígitos). Linhas com documento inválido aparecem destacadas em vermelho.
  • Você pode colar várias vezes — novos lançamentos são acrescentados aos já importados.

Identificação para Conciliação

Prefixo gravado em cada lançamento (campo "Nº Documento") para facilitar conciliação no extrato.

Antes do primeiro envio

Arquivo gerado conforme Manual CNAB 240 Pagamentos Inter v1.21 (10/02/2026), modalidade Pix por chave CPF/CNPJ — forma de iniciação "03", tipo de serviço "98" (Diversos), forma de lançamento "45" (Pix). O tipo de documento (1=CPF, 2=CNPJ) é detectado automaticamente pelo tamanho. Cada favorecido precisa ter o CPF/CNPJ cadastrado como chave Pix no banco dele.

Recomendo importar em homologação Inter uma vez para confirmar antes do primeiro envio em produção.
